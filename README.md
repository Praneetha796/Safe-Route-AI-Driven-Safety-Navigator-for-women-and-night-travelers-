SafeRoute is an AI-driven safety navigation system designed for women and night travelers. It provides real-time GPS tracking, safe route suggestions, danger zone detection, voice-based emergency activation, and live location sharing to enhance personal safety.

 ## Live Demo
   Project Link:
   https://praneetha796.github.io/Safe-Route-AI-Driven-Safety-Navigator-for-women-and-night-travelers-/
   click the link to use the application in real-time.

## Features
- Real-time GPS location tracking
- Map visualization using Leaflet.js
- Basic safe route generation
- Voice assistant for hands-free commands
- Emergency SOS activation
- Emergency contact management
- Live location sharing via Google Maps link
- System log for activity tracking

 ## Technologies Used
 - HTML
 - CSS (Tailwind CSS)
 - Javascript
 - Leaflet.js (Map rendering)
 - OpenstreetMap (Map data)
 - Geolocation API (Location tracking)
 - Web Speech API (Voice recognition)

 ## System Workflow
 1. The system fetches the user's real-time location using the Geolocation API.
 2. The map is diplayed using Leaflet.js.
 3. A route is generated and displayed on the map.
 4. The user can activate the SOS button manually or using voice commands.
 5. Emergency alerts are sent to saved contacts.
 6. The system generates a live location link for quick sharing.
 7. All activities are recorded in the system log.

 ## Voice Commands
 - "help" / "emergency" -> Activate SOS
 - "start" -> Start safe route navigation

  ## How to Run
  1. Download or clone this repository.
  2. Open the project folder.
  3. Run the 'index.html' file in Google Chrome.
  4. Allow:
        - Location access
        - Microphone access

   ## Objective
   To provide a simple and effective safety system that enables users to access emergency support quickly and navigate using voice commands without manual interaction.

 ## Limitations
 - Requires internet connection for maps.
 - Voice recognition works best in Chrome.
 - Route generation is basic (not real-time traffic based).
 - emegency alerts are simulated (not actual SMS)
